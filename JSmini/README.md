# JavaScript Mini Projects

This folder contains beginner-friendly JavaScript projects demonstrating core JavaScript concepts and DOM manipulation.

## Complete File List & What They Demonstrate

### File: a.js
**Purpose:** DOM manipulation - Show/Hide element demo
**Concepts Used:**
- `document.addEventListener()` - Event listener attachment
- `DOMContentLoaded` event - Ensuring DOM is ready
- `document.getElementById()` - Selecting elements by ID
- Event handling (`addEventListener`)
- Style manipulation (`element.style.display`)
- Display property values (`'none'`, `'block'`)
- Function expressions
- Arrow functions (implied by button handlers)

---

### File: a.html
**Purpose:** HTML interface for a.js
**Concepts Used:**
- HTML element structure
- Button elements (`<button>`)
- Paragraph with ID (`<p id="demo">`)
- Script tag for external JavaScript (`<script src="a.js">`)
- Basic page layout

---

## All JavaScript Concepts Used

### DOM Selection
- `document.getElementById()` - Get element by ID

### Event Handling
- `addEventListener()` - Attach event listeners
- `DOMContentLoaded` - Wait for DOM to be ready
- Event types: `click`

### DOM Manipulation
- Accessing element properties (`style`, `display`)
- Setting CSS properties programmatically
- Showing/hiding elements

### JavaScript Basics
- Variables (`const`)
- Functions
- Event handlers

## How to View

Open `a.html` in a web browser:

```bash
# Open in default browser (macOS)
open a.html

# Or drag and drop the file into your browser
```

## What You'll See

1. An image displayed on the page
2. Two buttons: "Hide" and "Show"
3. Clicking "Hide" removes the image
4. Clicking "Show" displays the image again

## Topics Covered

- DOM (Document Object Model) manipulation
- Event listeners
- Element selection
- Style manipulation
- Basic JavaScript syntax
