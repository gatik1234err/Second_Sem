## All C++ Concepts Used Across All Files

### Basic I/O
- `cout` - Standard output
- `cin` - Standard input
- `endl` - End line manipulator

### Data Types
- `int` - Integer
- `float` - Single precision float
- `double` - Double precision float
- `char` - Character
- `bool` - Boolean
- `long` - Long integer
- `string` - String of characters

### Operators
- **Arithmetic:** `+`, `-`, `*`, `/`, `%`
- **Assignment:** `=`, `+=`, `-=`, `*=`, `/=`
- **Comparison:** `==`, `!=`, `<`, `>`, `<=`, `>=`
- **Logical:** `&&` (AND), `||` (OR), `!` (NOT)
- **Ternary:** `condition ? value1 : value2`
- **Increment/Decrement:** `++`, `--`
- **sizeof:** Memory size operator
- **Address-of:** `&`
- **Dereference:** `*`

### Control Flow
- **if-else:** Conditional execution
- **switch-case:** Multi-way branching
- **while loops:** Condition-based iteration
- **for loops:** Counter-based iteration

### Functions
- Function declaration and definition
- Return types: `void`, `int`, `bool`
- Parameters: by value, by reference, by pointer
- Function calls
- Function overloading (conceptual)

### Object-Oriented Programming
- `class` definition
- **Access Specifiers:** `public`, `private`
- Member variables
- Member functions
- Constructors
- Object creation
- Member access (`.`)

### Advanced Concepts
- Recursion (function calling itself)
- Pointers (`*`)
- References (`&`)
- `const` member functions
- Constructor initialization lists
- Standard library functions (`std::max`)

### Libraries Used
- `<iostream>` - Input/output streams
- `<algorithm>` - Algorithm library (std::max)
- `<string>` - String operations
- `<cmath>` - Mathematical functions
- `<cstring>` - C-string operations
