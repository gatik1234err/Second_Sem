# HTML & CSS Learning Projects

This folder contains various HTML and CSS learning projects demonstrating different web development concepts.

## Complete File List & What They Demonstrate

### File: a.html
**Purpose:** Tech Blog with semantic HTML structure
**Concepts Used:**
- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`)
- Link elements (`<a>`)
- External CSS stylesheet linking (`<link rel="stylesheet" href="styles.css">`)
- Inline CSS styling (`style="color: blue; font-size: 30px;"`)
- Headings (`<h1>`, `<h2>`)
- Paragraphs (`<p>`)
- Classes for styling (`.highlight`)

---

## All HTML Concepts Used

### Semantic Elements
- `<header>` - Page or section header
- `<nav>` - Navigation links
- `<main>` - Main content area
- `<article>` - Self-contained content
- `<section>` - Thematic grouping
- `<aside>` - Sidebar content
- `<footer>` - Page or section footer

### Text Elements
- Headings (`<h1>` to `<h6>`)
- Paragraphs (`<p>`)
- Lists (`<ul>`, `<ol>`, `<li>`)
- Links (`<a>`)
- Images (`<img>`)

### Text Formatting
- `<b>` - Bold
- `<i>` - Italic
- `<u>` - Underline
- `<s>` - Strikethrough
- `<mark>` - Mark/highlight

### Forms
- `<form>` - Form container
- `<input>` - Various input types
- `<label>` - Form labels
- `<select>` - Dropdown
- `<option>` - Dropdown options
- `<textarea>` - Multi-line text
- `<button>` - Buttons

### Form Attributes
- `type` - Input type
- `id` - Unique identifier
- `name` - Form field name
- `placeholder` - Hint text
- `required` - Required field
- `value` - Default value

### CSS Selectors
- Element selectors (`h1`, `p`, etc.)
- Class selectors (`.highlight`, `.form-group`)
- Pseudo-class selectors (`nth-child`)

### CSS Properties
- Colors (`color`, `background`)
- Typography (`font-weight`, `text-decoration`)
- Spacing (`padding`, `margin`)
- Layout (`text-align`, `max-width`)
- Effects (`border-radius`, `box-shadow`)
- Backgrounds (`linear-gradient`)
