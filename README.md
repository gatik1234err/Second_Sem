## Getting Started

### For C++ Projects
```bash
# Navigate to Cpp folder
cd Cpp

# Compile any program
g++ filename.cpp -o output_name

# Run the program
./output_name
```

### For HTML/CSS/JS Projects
Simply open any `.html` file in a web browser:

```bash
# Open in default browser (macOS)
open filename.html

# Or drag and drop the file into your browser
```

### For htmlmini Portfolio
```bash
# Navigate to htmlmini folder
cd htmlmini

# Open index.html in a browser
open index.html
```

## Learning Path

### Beginner:
1. Start with **Cpp/a.cpp** - Basic C++ concepts
2. Move to **HTML/c.html** - Basic HTML text formatting
3. Try **JSmini/a.js** - Simple JavaScript DOM manipulation

### Intermediate:
4. Study **Cpp/o.cpp** through **Cpp/z.cpp** - Loops and functions
5. Explore **HTML/forms.html** - Form creation and styling
6. Build **htmlmini/** - Complete website project

### Advanced:
7. Learn **Cpp/t.cpp** through **Cpp/w.cpp** - Object-oriented programming
8. Master recursion with **Cpp/z.cpp** - Recursive Fibonacci
9. Understand pointers with **Cpp/w.cpp** - Call by value/reference/pointer
